159 /*!******************************************************************************

161  * transcoder devices. This is a high level API function which is used
162  * mostly with user application like FFMpeg that relies on those resources.
163  * In case of custom application integration, revised functionality might
164  * be necessary utilizing coresponding API functions.
165  *
166  * \param[in] should_match_rev 0: transcoder firmware revision matching the

170  * or fail; else: timeout will fail init once reached

175  *
176  *******************************************************************************/
