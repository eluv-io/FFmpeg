◆ remove_substring_pattern()

uint32_t remove_substring_pattern ( char *  main_str,
const char *  pattern 
)

Remove a string-pattern from a string in-place.

Parameters
[in,out]main_strNull terminated array of characters to operate upon in-place.
[in]patternNull terminated array of characters to remove from main_str. Supports special characters '#' and '+' for digit matching and repeated matching respectively. Note, there is no way to escape the special characters. Example: char a_str[10] = "aaa123qwe"; char b_str[5] = "a+#+"; remove_substring_pattern(a_str, b_str); printf("%s\n", a_str); Output: qwe
Returns
If characters removed, returns 1 If no characters removed, returns 0
